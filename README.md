# Facebook Clone

A clean, mobile-first Facebook UI clone built with HTML, CSS, and JavaScript.
Designed to look and feel like the real Facebook on both desktop and mobile.

## Pages

- News feed with stories, posts, and create post box
- Profile page with cover photo, posts grid, and friends section
- Friends page

## Features

- Fully responsive, mobile-first layout
- Bottom navigation bar on mobile
- Horizontal scrollable stories row
- 19 posts with images, reactions, and expandable comments
- Like button toggles blue on click
- Messenger panel (desktop)
- Right sidebar with contacts, sponsored, and birthdays (desktop)
- Left sidebar with navigation and shortcuts (desktop)
- Dark overlay story cards with friend names
- Sponsored section with thumbnails

## Tech Stack

- HTML
- CSS
- JavaScript
- Font Awesome 6 (icons)
